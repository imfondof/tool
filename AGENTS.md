# OpenSpec 指令

这些指令用于在本项目中与 AI 助手协作。

当请求满足以下条件时，必须打开 `@/openspec/AGENTS.md`：
- 提到规划或提案（如 proposal、spec、change、plan）
- 引入新能力、破坏性变更、架构调整，或较大的性能/安全工作
- 描述含糊，需要在编码前参考权威规格

使用 `@/openspec/AGENTS.md` 了解：
- 如何创建和应用变更提案
- 规格格式与约定
- 项目结构与指南

请保留该受管块，便于 `openspec update` 更新指令。
